# Shooter
- This shoot enemies game is a simple and entertaining game built using JavaScript, HTML canvas, and the GSAP (GreenSock Animation Platform) library. The game is set in a futuristic world where the player must defend their territory from incoming enemy ships. The objective of the game is to use a spacecraft to shoot down as many enemy ships as possible.

The game is designed using HTML canvas, which provides a dynamic and visually appealing environment for players to enjoy. The animations in the game are created using the GSAP library, which provides smooth and seamless animations, bringing the game to life.
